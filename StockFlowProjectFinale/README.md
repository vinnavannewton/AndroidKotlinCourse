# StockFlow

Compose Multiplatform stock trading demo for desktop and Android.

## Desktop

```bash
./gradlew run
```

## Linux package

```bash
./gradlew packageDeb
./gradlew packageRpm
```

## Windows package

```bash
./gradlew packageMsi
./gradlew packageExe
```

## Android

```bash
./gradlew assembleDebug
```
